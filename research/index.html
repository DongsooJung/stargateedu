Overview

연구용 데이터 축적을 자동화합니다

인증키 이중 인코딩, XML 오류 응답, 페이지네이션, 호출 제한처럼 공공데이터포털에서 자주 발생하는 문제를 공통 클라이언트에서 처리합니다.

01

다중 API 수집

소상공인 상권정보, 국토부 실거래가, 기상청 단기예보 등 서로 다른 응답 구조를 통합합니다.

02

안전한 인증 처리

serviceKey의 %2F·%2B·%3D 재인코딩을 방지하여 403 오류를 줄입니다.

03

연구 데이터 적재

수집 결과를 시점별 CSV로 저장하고 Notion 연구 로그와 연결할 수 있도록 설계했습니다.

현재 제한사항범용 수집기 public_data_collector.py는 아직 공통 클라이언트 전환 전 상태입니다. 인코딩된 인증키를 사용할 때 403이 발생할 수 있으므로 우선 sbiz_collector.py와 preflight.py를 사용하세요.
Quick Start

로컬 또는 연구 서버에서 실행

GitHub Pages는 Python을 실행하지 않으므로 이 페이지는 프로젝트 문서 허브 역할을 합니다. 실제 수집은 Windows 11, Linux 서버, GitHub Actions 또는 n8n 실행환경에서 수행합니다.

bash · 설치 및 테스트
git clone https://github.com/DongsooJung/stargateedu.git
cd stargateedu/research/public-data-collector
python -m venv .venv
# Windows: .venv\Scripts\activate
# Linux/macOS: source .venv/bin/activate
pip install -r requirements.txt

export DATA_GO_KR_API_KEY="발급받은키"
python preflight.py --dump-schema
python sbiz_collector.py --once
python -m pytest tests/ -q
